طريقة التشغيل باستخدام XAMPP:

1. افتح مجلد XAMPP (عادة يكون: C:/xampp/htdocs)
2. انسخ مجلد المشروع داخل htdocs (مثلاً: kanban-board)
3. شغّل Apache من XAMPP Control Panel
4. افتح المتصفح على:
   http://localhost/kanban-board/

المشروع يستخدم:
- HTML / CSS / JavaScript
- PHP لتخزين المهام في ملف JSON
